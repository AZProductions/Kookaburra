Features

Feature rich interpreter.

Kookaburra's interpreter has a lot of pro level features, such as Element binding.  Create programs in seconds. Read more..

Powerful CLI.

Kookaburra's CLI has pro level features, like Advanced Download and Password Generation. Read more..

Cross-Platform environment adaptation.

No matter your device, Kookaburra adapts to it. On Windows you can edit files using Notepad, on Linux you can use Vim or Nano.

Support for 3rd party terminals. 

Kookaburra supports 3rd party terminals like Windows Terminal out of the box. No tweaking is needed, you automatically get the best results. 

Easy to understand

Kookaburra is a simple programming language, designed for beginners but
with sufficient complexity for more advanced coders. Read more..

Intuitive commands.

Kookaburra's commands are fine-tuned to perfection. Create advanced scripts and get the performance you need.
